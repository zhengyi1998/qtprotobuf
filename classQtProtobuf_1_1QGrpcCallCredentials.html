Detailed Description

The QGrpcCallCredentials is base class for gRPC call credentials. You may inherrit of this calls to create you own credentials implementation, that will be used for each method call. Classes inherited of QGrpcCallCredentials should reimplement QtProtobuf::QGrpcCredentialMap operator() method and return actual credentials map for the call.
